Maps, bikes, and general nerdiness.

Hello world!

Welcome to my humble home page. I'm a newbie developer, though I've been dabbling with programming here and there since grade school. I work as a GIS Intern at TriMet, where I assist with the Regional Trip Planner project, an open source multi-modal trip planner built with OpenTripPlanner. You can generally find me editing OpenStreetMap or riding one of my bikes around Portland, OR.

Side projects

  • A new version of my CivicApps-award-winning Portland Bike to Transit Map
  • Nifty neon-colored PDX bike route map tiles [retina tile version] that I made with TileMill and MapBox


  • The same Portland-area bike routes [retina tile version], with a MapBox streets + terrain background


  • A test/development version of a mobile web app for doing a curb ramp inventory, a project for the Willamette Pedestrian Coalition GIS Jam [GitHub project link]

Stuff I heart

  • OpenPlans: my favorite nonprofit and the lead organization for the OpenTripPlanner project
  • Quantum GIS: an increasingly awesome open-source alternative to ArcGIS
  • PostgreSQL with PostGIS: spatially enabled database system that can also do many spatial analysis + data processing tasks quickly and easily with SQL
  • OpenGeo: origin of PostGIS, OpenLayers, GeoServer, and other great stuff
  • CartoDB: PostGIS in the cloud, easy to access and control with web apps
  • Codecademy: a great way to start learning to code
  • PDX OSGEO: Portland open source geospatial software user group
  • My friends at the Portland Python User Group
  • BikePortland - all the PDX bike news that's fit to blog
  • Transit Maps Tumblr: so many pretty maps!
  • More links coming soon!
